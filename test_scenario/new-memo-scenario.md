# E2E 테스트 시나리오: 새 메모 작성

## 목적

사용자가 새 메모를 성공적으로 작성하고 저장할 수 있는지 확인합니다.

## 시나리오 단계

1. "새 메모" 버튼을 클릭합니다.
2. "새 메모 작성" 폼이 나타나는지 확인합니다.
3. 제목 입력란에 "테스트 메모 제목"을 입력합니다.
4. 카테고리 드롭다운에서 "업무"를 선택합니다.
5. 내용 입력란에 "테스트 메모 내용입니다."를 입력합니다.
6. 태그 입력란에 "테스트"를 입력하고 "추가" 버튼을 클릭합니다.
7. "저장하기" 버튼을 클릭합니다.
8. 메모 목록에 새로 추가된 "테스트 메모 제목"이 보이는지 확인합니다.
